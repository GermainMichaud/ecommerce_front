## Getting starting

### Clone the repository

```sh
git clone git@github.com:GermainMichaud/ecommerce_front.git
cd ecommerce_front/
```

### Install dependencies

```sh
# with npm (default for this project)
npm install

# with yarn
yarn

# with pnpm
pnpm install
```

### Run the project in `DEV` mode

```sh
# with npm
npm run dev

# with yarn
yarn dev

# with pnpm
pnpm dev
```

### Run the project in `PRODUCTION` mode

```sh
# with npm
npm run build
npm run preview

# with yarn
yarn build
yarn preview

# with pnpm
pnpm build
pnpm preview
```

### Run tests

```sh
# with npm
npm test

# with yarn
yarn test

# with pnpm
pnpm test
```
